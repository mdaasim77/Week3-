Q6: List any 5 semantic tags in hTMl along with their descriptions

Here are five commonly used semantic tags in HTML along with their descriptions:

  1. <header>: The <header> element represents the introductory content or a container for a group of introductory content at the beginning of a section or page. It typically includes the site or section title, logo, navigation, or other header-related elements.
  2. <nav>: The <nav> element represents a section of a page that contains navigation links. It is used to define a navigation menu or a set of navigation links that allow users to navigate within the website or to different sections of the page.
  3. <main>: The <main> element represents the main content area of a document or a section of a document. It should contain the primary content that is unique to the document or section, excluding headers, footers, navigation, or sidebars.
  4. <section>: The <section> element represents a thematic grouping of content within a document or a section of a document. It is often used to divide a page into different sections or topics, providing a logical and structural organization of content.
  5. <footer>: The <footer> element represents the footer of a document, section, or article. It typically contains information such as copyright notices, author attribution, contact information, related links, or other metadata relevant to the content or the website.

These semantic tags provide a structural and meaningful representation of the content, making it easier for search engines, screen readers, and other assistive technologies to understand and navigate the web page. By using these semantic tags appropriately, you enhance the accessibility, usability, and maintainability of your HTML documents.
