Q3: What is web accessibility ? list some of the assistive devices which play a major role in providing accessibility .

Web accessibility refers to the inclusive practice of designing and developing websites and web content in a way that ensures equal access and usability for all users, including people with disabilities. It aims to eliminate barriers that prevent individuals with disabilities from effectively navigating, perceiving, interacting, and contributing to the web.

  • Screen Readers: Screen readers are software applications that read aloud the text on a web page, making it accessible to individuals with visual impairments. They provide auditory output and allow users to navigate the web using keyboard commands.
  • Screen Magnifiers: Screen magnifiers are tools that enlarge the content displayed on a computer screen, helping individuals with low vision to read and view web pages more easily.
  • Braille Displays: Braille displays are devices that convert text from a web page into Braille output. They allow individuals who are blind or have severe visual impairments to read and navigate the web using Braille characters.
  • Voice Recognition Software: Voice recognition software, also known as speech recognition software, allows users to interact with a website using voice commands instead of typing or using a mouse. It is particularly beneficial for individuals with mobility impairments or those who have difficulty using a keyboard or mouse.
  • Switch Devices: Switch devices are assistive devices that enable individuals with limited motor skills to interact with the web. These devices allow users to perform various actions, such as clicking, scrolling, or navigating through web pages, using simple switches or buttons.
