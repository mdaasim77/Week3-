What is HTML

HTML stands for HyperText Markup Language. It is the standard markup language used for creating and structuring the content on webpages. HTML uses a set of tags to define the elements and structure of a webpage, such as headings, paragraphs, images, links, lists, tables, forms, and more.

What is Semantic HTML

Semantic HTML refers to the practice of using HTML tags that accurately and meaningfully describe the structure and purpose of the content within a webpage. Instead of using generic or presentational tags, semantic HTML focuses on choosing tags that convey the semantic meaning of the content.

Commonly used Semantic Tags

<header>, <nav>, <main>, <section>, <article>, <aside>, and <footer>
